# pr0gramm

This is the pr0gramm App in SwiftUI for all iOS Devices running iOS 14 and above.

## Features
- View the feed in New and Top

## Planned and in the works
- Login
- Viewing tags and comments
- Vote from Posts
- Switching between SFW, NSFW and NSFL
